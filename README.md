Sistemas de Notas de Alunos

- Esse projeto é um Sistema de Cadastro de Notas de Alunos, desenvolvido em Java com Swing / JFrame.
- O sistema possui login com validação, um menu principal e um módulo para cadastro de alunos e notas, exibindo no final a média e sua situação "Aprovado" ou "Recuperação".

Esse projeto tem como conceito a aplicação de:
- Programação Orientada a Objetos (POO)
- Interface Gráfica com Java Swing
- Validação de dados
- Lógica de cálculo de média
- Organização de telas e menus
- Controle de versão com Git e GitHub

Obs: esse projeto foi desenvolvido exclusivamente para fins acadêmicos, como parte do trabalho da disciplina. Contudo, sugestões de melhoria são bem-vindas.
